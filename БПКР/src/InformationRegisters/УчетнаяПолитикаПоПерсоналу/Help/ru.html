Учетная политика по персоналу

Сведения об учетной политике по персоналу хранятся в периодическом регистре сведений «Учетная политика по персоналу».

В рамках конфигурации в учетной политике по персоналу предусмотрены такие опции как:


Закладка «Общие»

  • Возврат ЗП отрицательной суммой – сторно запись, формирование проводки возврата от сотрудника отрицательной суммой.
  • Пределы по подразделениям. При установленном флажке становится видимой закладка «Пределы налогов», на которой для каждого подразделения указываются Минимальный расчетный доход, ГНС, Среднемесячная зарплата по СФ и отделение соцфонда. При установленном флажке «Налоги по подразделениям», но не заполненной вкладке «Пределы налогов» по менее чем 2м подразделениям, флажок будет сброшен;



Закладка «П/налог»

  • Минимальный расчетный облагаемый доход (МРД). В соответствии со ст. 163 НК КР «Размер минимального расчетного дохода в месяц устанавливается уполномоченным налоговым органом ежегодно на основании данных статистического органа Кыргызской Республики по региональной среднемесячной заработной плате работников по состоянию на 1 октября предыдущего года с учетом стандартных вычетов, предусмотренных пунктами 1 и 3 статьи 170 настоящего Кодекса…».
  • Доплата ПН за счет фирмы. Если флажок установлен, то сумма подоходного налога, доплачиваемая до значения, соответствующего минимальному расчетному доходу, относится на расходы организации. Если флажок не установлен, то доплата производится за счет работника;
  • Счет учета расходов по доплате ПН. Если установлен флажок «Доплата ПН за счет фирмы», то доплачиваемая сумма подоходного налога, относится в дебет указанного счета;
  • Статья по доплате ПН. Если установлен флажок «Доплата ПН за счет фирмы», то доплачиваемая сумма подоходного налога относится на указанную статью;
  • Счет учета по ПН для прочих физических лиц.
  • Расчет ПН без накопления. – При установленном флажке расчет ПН за месяц.



Закладка «Соцотчисления»

  • Счет учета расчетов ПФР - бухгалтерский счет учета расчетов с ПФ (3531). Назначается как кредитовый счет проводок по начислению отчислений в пенсионный фонд с фирмы;
  • Счет учета расчетов ГНПФР - бухгалтерский счет учета ГН ПФР (3534). Назначается в проводках по начислению отчислений в пенсионный фонд с работника;
  • Счет учета расчетов ПФФ - бухгалтерский счет учета ПФФ (3531). Назначается в проводках по начислению отчислений в пенсионный фонд с фирмы;
  • Счет учета расчетов МСФ - бухгалтерский счет учета расчетов с ФОМС (3532). Назначается как кредитовый счет проводок по начислению в ФОМС;
  • Счет учета расчетов ФОТФ - бухгалтерский счет учета расчетов с ФОТФ (3533). Назначается как кредитовый счет проводок по начислению отчислений в пенсионный фонд с фирмы;
  • Счет учета штрафы, пени, неустойки по СФ
  • Среднемесячная заработная плата (СФ). Ставка среднемесячной заработной платы меняется ежегодно с 1 июля текущего года. После изменения ставки, необходимо оформить новую учетную политику, указав размер новой ставки СЗПСФ.
  • Процент от СЗП. Указывается размер процента от среднемесячной заработной платы.
  • Счет учета расходов по корректировке (СФ). Суммы доплат по соцотчислениям с фирмы относятся в дебет указанного счета;
  • Статья по корректировке (СФ). Указывается статья расходов, выбираемая из справочника «Статьи затрат»
  • Дата ограничения начисления ГНПФ (мужчины): Указывается дата возрастного ограничения, для отчета по отчислениям в Соц.фонд.
  • Дата ограничения начисления ГНПФ (женщины): Указывается дата возрастного ограничения, для отчета по отчислениям в Соц.фонд.
  • Количество сотрудников для отчета СФ – метод определения количества работников для отчетов по соцотчислениям (по приказам или по начислениям).



Закладка «Дополнительно»

  • Ставка корп. отчислений – указывается ставка в процентах (например, 1%). Указывается, если все работники уплачивают одинаковый процент корпоративных отчислений. Дополнительно при оформлении приказов для работников на закладке «Работники» должен быть указан признак «Корп.отч»;
  • Проф.взнос за вычетом ПН. Если флажок установлен, то при расчете корпоративных отчислений будет отнят размер подоходного налога работника в расчетном периоде;
  • Проф.взнос за вычетом ПФР. Если флажок установлен, то при расчете корпоративных отчислений будет отнят размер пенсионного фонда работника в расчетном периоде;
  • Счет учета расчетов (Проф. взнос). Указывается счет расчета по профсоюзным отчислениям.
