Учетная политика организаций

Действующее законодательство предоставляет возможность выбора одного из нескольких способов ведения бухгалтерского и налогового учета по ряду направлений. Совокупность выбранных способов учета является учетной политикой организации

Сведения об учетной политике организации по бухгалтерскому и налоговому учету хранятся в регистре сведений «Учетная политика». Каждая запись этого регистра сведений характеризует учетную политику организации за определенный отчетный период (год).

При вводе новой записи об учетной политике необходимо указать организацию, если учет ведется по нескольким организациям; дату начала отчетного периода, в котором будет применяться данная учетная политика.

Форма нового элемента регистра сведений «Учетная политика» содержит несколько разделов, на которых указываются параметры учетной политики по бухгалтерскому и налоговому учету.

Раздел «Налоги»

В разделе «Налоги» устанавливаются параметры учета доходов и расходов в целях налогообложения прибыли:

  • Признак плательщика налога на добавленную стоимость, если организация зарегистрирована как плательщик НДС;
  • Признак «Контракт» - при установленном флажке:

- в отчете по НДС будут заполняться приложения 4 и 5 (на основе налогового контракта), вместо 1 и 2 приложений, соответственно данные будут выводиться в основную форму отчета по НДС;
- в отчете по НСП заполняется подраздел Г. Реализация товаров (работ, услуг), налоговое обязательство по которой исполняется на основе налогового контракта.

  • Признак плательщика налога с продаж;
  • Признак плательщика налога на прибыль;
  • Признак плательщика налога на имущество;
  • Признак плательщика единого налога;
  • Учет НДС на авансы - при установленном флажке НДС на авансы будут учитываться на отдельном счете 1549:
  • Указывать признак зачета НДС. При установленном флажке в документах поступления (Поступление товаров и услуг, Возврат товаров и услуг поставщику), для каждой позиции номенклатуры указывается вид зачета НДС (зачет, себестоимость или распределение), для незаполненных позиций и для табличной части ОС признак принимает значение «Зачет». Если флажок в учетной политике не установлен, то признак «Зачет НДС» скрыт и принимает значение по умолчанию «Зачет»;
  • Не учитывать закупки без НДС. При установленном флажке закупки без НДС не будут показываться в отчете по НДС;
  • Отчет по НДС по поставке:
  • Ставка НСП для расчета НДС на авансы. По умолчанию – «Торговля», при необходимости можно выбрать другую ставку из справочника «Ставки НСП»;
  • Ставка налога на прибыль;
  • Пороговый % освобожденных поставок. По умолчанию 5%, можно менять;
  • Ставка таможенного сбора. По умолчанию 0,15%, можно менять.

Раздел «Основные средства»

В разделе «Основные средства» настраиваются следующие параметры учета:

Флажок «Начисление амортизации ОС по месяцам». Если флажок снят, то расчет амортизации ОС начинается со дня ввода в эксплуатацию, пропорционально количеству дней эксплуатации в месяце принятия к учету или другой операции по движению ОС.

Минимум стоимости ОС для НУ, признаваемая для ОС по НК в соответствии с НК КР (статья 199 п.1)

Раздел «Товары»

В разделе устанавливаются параметры учета товаров.

Способ оценки товаров, выбор из списка:

• По средней (СВЦ)
• ФИФО;

Ставка НСП для реализации (закладка «Товары») по умолчанию устанавливается «Торговля», можно менять;

Ставка НСП для реализации (закладка «Услуги») по умолчанию устанавливается «Прочее», можно менять

Раздел «МБП»


Вид учета износа МБП.Выбор из списка:

• 100% при вводе
• 100% при списании.

Раздел «Единый налог»


При установлении признака «Плательщик ЕН» становится видимым раздел «Единый налог».

В данном разделе расписаны следующие виды деятельности и ставки применяемые для них:

• Вид деятельности «Переработка, производство, торговля
Ставка (наличная форма) по умолчанию 4, можно менять;
Ставка (безналичная форма) по умолчанию 6, можно менять.

• Вид деятельности «Прочее»
Ставка (наличная форма) по умолчанию 2, можно менять;
Ставка (безналичная форма) по умолчанию 3, можно менять.
