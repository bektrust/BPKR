Трудовое соглашение

Документ Трудовое соглашение предназначен для оформления трудовых отношений между работодателем и наемным сотрудником за оплату его работы по определенной специальности, квалификации или должности.

На каждое трудовое соглашение оформляется один документ. Трудовое соглашение оформляется на период не больше одного месяца.

В документе Трудовое соглашение вручную выбираются Исполнитель, Подразделение, Должность, Статус из соответствующих справочников.

В полях Дата приема и по указываются даты начала и окончания работы.

В поле Размер указывается сумма оплаты.

В поле Вид начисления указывается вид начисления, по которому нужно отразить оплату по трудовому соглашению (способ расчета – только фиксированной суммой).

В поле Способ отражения указывается отражение способа начисления.

На закладке Дополнительно указывается комментарий к документу.


Печатные формы:


Для документа Трудовое соглашение предусмотрена печатная форма:
