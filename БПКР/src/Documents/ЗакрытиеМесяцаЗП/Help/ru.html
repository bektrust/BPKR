Закрытие месяца по ЗП

Документ предназначен для расчета подоходного налога, соц.отчислений с работников и с организации, плановых удержаний, а также удержаний за период. Документ оформляется один раз в месяц по организации в целом. Этим же документом формируются все проводки по начислениям и удержаниям по организации.

На всех закладках документа доступно ручное редактирование, которое можно сохранить при закрытии документа.

В шапке документа необходимо выбрать период.

По кнопке Выполнить закрытие месяца формируются две закладки документа: Налоги по заработной плате и Удержания.

На закладке Налоги по заработной плате для каждого работника организации приведены итоговые начисления и налоги.

На закладке Удержания приведены все плановые удержания, а также удержания за период.
По кнопке Добавить можно добавить новую строку, и заполнить ее новым удержанием вручную.
.
На закладке Дополнительно указывается комментарий к документу.

В подвале документа выходят все суммированные итоги табличных данных.

По гиперссылке Журнал проводок бухгалтерского учета приведены все сформированные по организации проводки по начислениям и удержаниям.
