Текущий расчет налога на прибыль

Документ предназначен для расчета квартальных предварительных платежей по налогу на прибыль.

Ежеквартально (за исключением 4-го квартала) создается документ, в котором производится расчет налога на прибыль с начала года по конец заданного квартала.

При проведении производится контроль – должен быть только один документ по данной организации за данный квартал.

Пользователем задается только дата квартала.

По кнопке Рассчитать производиться весь расчет.

По кнопке Расшифровка выходит Отчет по текущему налогу на прибыль, где расписаны все Доходы и Расходы (БУ). Данный отчет доступен для печати.

Флажок Ручная корректировка обеспечивает доступность реквизитов для редактирования. При снятом флажке эти реквизиты недоступны.

На основании документа можно выписать Платежное поручение (исходящее).
