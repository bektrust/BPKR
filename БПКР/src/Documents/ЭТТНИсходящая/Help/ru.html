Электронная товарнотранспортная накладная (исходящая)

Электронная товаротранспортная накладная (далее ЭТТН) – это электронный документ, который создается продавцом товаров в информационной системе ЭТТН (далее ИС ЭТТН).

Этот документ может создаваться в ИС ЭТТН пользователем вручную или загружаться из файла XML, который можно сформировать с помощью 1С. Для этого в 1С предназначен документ Электронная товаротранспортная накладная (исходящая).
Документ создается на основании:

  • Реализации товаров и услуг;
  • Возврата товаров и услуг от покупателя;
  • Списания товаров;
  • Перемещения товаров;
  • Оприходования товаров;
  • Отчета производства за смену.

В документе предусмотрены следующие виды операций:

Операция «Реализация»

Операция «Перемещение»

Операция «Списание»

Операция «Оприходование»

Операция «Возврат»

Операция «Выгрузка остатков»

Операция «Реализация»

Операция «Реализация» – на основании документа Реализация товаров и услуг. Далее с помощью сканера штрих кодов или терминала сбора данных необходимо заполнить коды маркировки в табличной части Код маркировки на закладке Товары.
Затем перейти на закладку Доставка и заполнить необходимые поля.

Обязательные для заполнения поля – закладка Товары:

  • Номенклатура;
  • Ед. измерения;
  • Количество;
  • Цена;
  • Сумма;
  • Масса груза (Нетто);
  • Масса груза (Брутто);
  • Код маркировки (если товар подлежит Маркировке).

Внимание! Для установки признака маркируемой продукции в карточке номенклатуры необходимо установить флажок «Это маркированная продукция».

Обязательные для заполнения поля – закладка Доставка:

  • Автомобиль;
  • Номер АТС;
  • Водитель;
  • Способ перевозки;
  • Перевозчик;
  • ИНН перевозчика.

Операция «Перемещение»

Операция «Перемещение» - на основании документа Перемещение товаров - автоматически заполнится табличная часть Товары. Затем необходимо перейти на закладку Доставка и заполнить необходимые поля. Далее нажать на кнопку Сохранить в формате xml и указать путь для сохранения файла. Данный файл в последующем следует загрузить в ИС ЭТТН.

Обязательные для заполнения поля – закладка Товары:

  • Номенклатура;
  • Ед. измерения;
  • Количество;
  • Цена;
  • Сумма;
  • Масса груза (Нетто);
  • Масса груза (Брутто).

Обязательные для заполнения поля – закладка Доставка:

  • Автомобиль;
  • Номер АТС;
  • Водитель;
  • Способ перевозки;
  • Перевозчик;
  • ИНН перевозчика.

Операция «Списание»

Операция «Списание» - на основании документа Списание товаров - заполнится табличная часть Товары, затем в шапке документа необходимо заполнить поле Причина списания. Причину списания следует выбрать из справочника Причины списания. Справочник заполняется через кнопку Подбор из классификатора. После заполнения причины списания нажать кнопку Записать.
Далее нажать на кнопку Сохранить в формате xml и указать путь для сохранения файла. Данный файл в последующем следует загрузить в ИС ЭТТН.

Обязательные для заполнения поля – закладка Товары:

  • Номенклатура;
  • Ед. измерения;
  • Количество;
  • Цена;
  • Код маркировки (если товар подлежит Маркировке).

Операция «Оприходование»

Операция «Оприходование» - на основании документа Оприходование товаров - заполняется табличная часть Товары, затем в шапке документа необходимо заполнить поле Причина оприходования. Причину оприходования следует выбрать из справочника Причины оприходования. Справочник заполняется через кнопку Подбор из классификатора. После заполнения причины оприходования нажать кнопку Записать.
Далее нажать на кнопку Сохранить в формате xml и указать путь для сохранения файла.

Обязательные поля – закладка Товары:

  • Номенклатура;
  • Ед. измерения;
  • Количество;
  • Цена.

Операция «Возврат»

Операция «Возврат» - на основании документов Возврат товаров и услуг от покупателя и Возврат товаров и услуг поставщику- заполняется табличная часть Товары.

Обязательные поля – закладка Товары:

  • Номенклатура;
  • Ед. измерения;
  • Количество;
  • Цена.

Операция «Выгрузка остатков»

Для того чтобы выгрузить из 1С остатки на складах необходимо оформить документ Электронная товаротранспортная накладная (исходящая) (раздел ЭТТН – Документы) с видом операции Выгрузка остатков.

В открывшейся форме выбрать склад, с которого нужно выгрузить остатки в поле Склад и указать Причину оприходования - Остаток на складе (причину оприходования следует выбирать из справочника Причины оприходования, который следует заполнить с помощью кнопки Подбор из классификатора). Затем в табличной части (далее "ТЧ") Товары нажать кнопку Заполнить по остаткам. ТЧ документа заполнится автоматически. После этого следует нажать кнопку Записать и нажать кнопку Сохранить в формате xml. Далее указать путь для сохранения файла. Затем файл следует загрузить в ИС ЭТТН.

Как найти: ЭТТН → Документы → Исходящая ЭТТН
