Больничный лист

Документ предназначен для оформления больничного сотруднику. Создается на каждого сотрудника отдельно.

Создать новый документ по кнопке Создать или в списке документов правой кнопкой мыши и выбрать Создать.

При создании нового документа автоматически устанавливается дата документа, можно отредактировать вручную.

Выбрать сотрудника, на которого оформляем больничные листы выбирается из справочника физические лица.

В поле График при выборе сотрудника автоматически подставляется график, указанный в приеме на работу (или кадровом перемещении).

Поле Стаж заполнится автоматически после выбора сотрудника по документу прием на работу, оно нужно для тоже чтобы определить проценты выплаты больничных.

Для сотрудников отработавших:

  • От 0 до 5ти лет оплачиваются 60%;
  • От 5ти до 8ми лет оплачиваются 80%;
  • От 8ми и выше оплачиваются 100% больничных.


В поле Метод расчета выбрать метод расчета по стрелке Показать все.
В методе расчета указать неполные месяцы:

  • Не изменять – для расчета СЗПД выводить фактически отработанные часы и начисленную сумму;
  • Дополнять – для расчета СЗПД выводить норм часов и размер по основному виду начисления;
  • Отбрасывать – в расчете не учитывать, если есть неявки, больничные и отработал меньше нормы часов.


В методе расчета выбрать вид начисления:

  • За счет организации
  • За счет социального фонда


Вид начисления должен иметь способ расчета По среднему заработку для больничного, а также список видов начислений, входящих в базу расчета больничного (закладка Базовые виды расчетов).

В шапке документа заполняется период для расчета больничного.

Поле Дней заполниться автоматически при заполнения Периода.

В поле Дней организации заполнится автоматически при заполнении Периода, если количество дней по больничному листу не превышает 10 дней. Их оплачивает организация.

В поле Дней за счет СФ заполнится автоматически при заполнении Периода, количеством дней превышающих 10 дней по больничному листу. Их оплачивает социальный фонд.

В поле СЗПД автоматически заполнится размер среднемесячной заработной платы за день после заполнения закладки База больничных.

На закладке База больничных по кнопке Заполнить автоматически заполняются данные базовых месяцев (месяц, норма дней, отработано дней, базовая сумма). По кнопке Добавить можно добавить строку и заполнить данные вручную.

На закладке Начисления по кнопке Рассчитать автоматически заполняются данные и рассчитывается сумма больничного. По кнопке Добавить можно добавить строку и заполнить данные вручную.

На закладке Дополнительно указать комментарий к документу Больничные листы.

Больничные листы должны вводиться до проведения документов Начисление зарплаты для того, чтобы мог сработать механизм вытеснения (дни больничного должны вытеснить дни оклада).
