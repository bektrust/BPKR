Документ "Передача товаров" 

Документ предназначен для отражения операций по передаче собственных материалов стороннему переработчику для выпуска продукции, передачи заказчику продукции, выпущенной из давальческого сырья, для передачи комитенту товаров, закупленных по договору комиссии, а также для безвозмездной передачи, т.е. когда товары предоставляются другой стороне, без получения от нее платы или иного встречного предоставления.
