Платежное поручение (входящее)

Документ Платежное поручение входящее предназначен для учета поступлений безналичных денежных средств на банковский счет организации.

При вводе документа в шапке нужно указать следующие реквизиты, общие для всех операций:

  • Номер входящего документа;
  • Дата;
  • Вид операции;
  • Банковский счет - банковский счет организации. В справочнике обязательно должны быть указаны счет учета и валюта для данного банковского счета, которые однозначно принимаются для документа Платежное поручение входящее в целом.

На закладке Расшифровка платежа указывается:

  • Сумма платежа - сумма поступления денежных средств;
  • Статья ДДС.

На закладке Дополнительно заполняются реквизиты:

  • Счет учета - подставляется из выбранного банковского счета;
  • Счет расчетов - выбирается из регистра Счета расчетов с контрагентами;

Если валюта договора и валюта по банку не совпадают, становятся видимыми поля:

  • Курс взаиморасчетов - заполняется автоматически по курсу НБКР, но может быть изменен вручную при договорном курсе;
  • Сумма взаиморасчетов - рассчитывается автоматически. При изменении суммы вручную меняется курс взаиморасчетов;

Если нужно оформить оплату по разным статьям движения денежных средств, то в шапке документа используется кнопка Список.

Оплата от покупателя

Операция предназначена для отражения полученной от покупателя оплаты за товары, работы и услуги.

  • Контрагент - покупатель, который является плательщиком денежных средств;
  • Договор;
  • Статья ДДС;
  • Сумма платежа - сумма оплаты;

Возврат от поставщика

Операция предназначена для отражения возврата денежных средств от поставщика, раннее оплаченных ему.

  • Контрагент (поставщик) - поставщик, которому возвращаются денежные средства;
  • Договор - договор контрагента с видом договора С поставщиком;

Возврат от сотрудника

Операция выбирается при возврате сотрудником заработной платы.

Возврат от подотчетника

Операция выбирается при возврате сотрудником денежных средств, выданных в подотчет.

Расчеты по займам

Операция предназначена для отражения расхода безналичных денежных средств по кредитам и займам.

Прочий приход

Операция предназначена для оформления поступления денежных средств в случаях, не предусмотренных стандартными операциями, то есть является универсальной.

Печатные формы

Для документа Платежное поручение (входящее) печатные формы не предусмотрены.

Ввод на основании

Ввод на основании документа Платежное поручение (входящее) не предусмотрен.
