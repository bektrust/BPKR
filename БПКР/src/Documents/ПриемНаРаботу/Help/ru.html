Прием на работу


Документ предназначен для оформления приема на работу. Создается на каждого сотрудника отдельно.

На закладке Приказ заполняются сведения для приема на работу сотрудника: выбирается сотрудник, которого принимают на работу, указываются Подразделение, Должность, График, Статус из соответствующих справочников.

В поле Дата приема указывается дата приема сотрудника на работу.

В поле Дата по при необходимости заполняется дата окончания периода работы сотрудника.

Поле Испытательный срок заполняется вручную, если сотрудник принят с испытательным сроком. Если принимается на работу без испытательного срока, дату не устанавливают.

В поле Размер указывается размер оклада принимаемого сотрудника.

В поле Вид начисления указывается основной вид начисления зарплаты.

В поле Допвычеты указывается количество иждивенцев.

В поле Предыдущий стаж указывается общий стаж по трудовой деятельности (лет, месяцев, дней).

Если у сотрудника несколько видов начислений, то их добавляют на закладке Прочие начисления.

На закладке Удержания оформляются Плановые удержания с сотрудника, если таковые имеются.

На закладке Дополнительно указывается комментарий к документу.


Печатные формы


Для документа Прием на работу предусмотрены следующие печатные формы:

  • Приказ о приеме на работу
  • Трудовой договор
