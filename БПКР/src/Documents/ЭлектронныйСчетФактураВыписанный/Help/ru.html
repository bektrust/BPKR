Электронный счет-фактура (выписанный)

Документ предназначен для формирования счетов-фактур в электронном виде для Информационной системы электронных счетов-фактур (ИС ЭСФ).

Создается документ Электронный СФ несколькими способами:

  • Из документа Реализация товаров и услуг или Возврат товаров и услуг от покупателя по ссылке Создать ЭСФ для вида Исходящие;
  • Из монитора ЭСФ на закладке Документы по кнопке Создать ЭСФ для вида Исходящие;
  • Заполнение непосредственно в документе Электронный СФ;
  • Из обработки Загрузка ЭСФ для вида Входящие;
  • Из документа Поступление товаров и услуг для вида Входящие.


Документ состоит из трех закладок:

  • Реквизиты. Задаются параметры для отбора документов. При создании документа Электронный СФ для Возврата товаров и услуг от покупателя нужно установить признак Корректировочный. (Заполняется автоматически в случае, если документ был создан из Реализации товаров и услуг или Возврат товаров от покупателя).
  • Товары/Услуги. Табличная часть заполняется по кнопке Подбор. (Заполняется автоматически в случае, если документ был создан из Реализации товаров и услуг или Возврат товаров от покупателя). Для Входящих ЭСФ - заполнение автоматически.
  • Дополнительно. Ручной ввод комментария.


Ввод номера ЭСФ осуществляется автоматически по кнопке Загрузка номеров ЭСФ в форме списка документа.

После создания документа Электронный СФ необходимо нажать на кнопку Сохранить в xml, после чего сохранить электронный счет-фактуру в XML файл. (Возможно в форме документа и форме списка документа Электронный СФ и в Мониторе ЭСФ). Сохраненный XML файл нужно загрузить в ИС ЭСФ.

По кнопке Создать на основании, для вида ЭСФ Входящие создается документ Поступление товаров и услуг.

Как найти: ЭСФ → Документы → Электронный СФ
