Командировка
Документ оформляется для регистрации командировок и расчета командировочных (раздел Зарплата и кадры – Командировка).
Командировка может быть оформлена на одного или нескольких сотрудников одного подразделения.
Создать новый документ по кнопке Создать или в списке документов правой кнопкой мыши и выбрать Создать.

При создании нового документа автоматически присваивается номер и дата документа, дату можно отредактировать вручную.

В шапке документа "Командировка" вручную заполняются следующие реквизиты:

• Страна - выбирается из справочника "Страны мира".
• Город - выбирается из справочника "Города". Варианты выбора города:
1. Написание наименования вручную и выбор из списка возможных вариантов.
2. Выбор из справочника Города - по стрелке "Показать все".
• Подразделение - выбирается из справочника "Подразделения".
• Основание командировки - заполняется вручную.
• Период – указать период командировки.

На закладке "Сотрудники" по кнопке Заполнить в табличной части заполнится список сотрудников, у которых оформлены приказы по выбранному подразделению. По кнопке Добавить или Подбор можно выбрать сотрудника вручную.

Суточные, Проживание и Проездные заполняются по кнопке Заполнить автоматически из документа "Установка тарифов командировочных".

На закладке "РКО" по кнопке Сформировать РКО создается документ "Расходный кассовый ордер" с видом операции "Выдача подотчетнику" (на каждого сотрудника отдельно, если документ оформляется на список сотрудников).

На закладке "Дополнительно" заполняются:
• Содержание (для печати приказа);
• Основание (для печати приказа);
• Комментарий к документу.
Документ имеет несколько печатных форм:
• Приказ
• Командировочное удостоверение
• Платежная ведомость
• Распоряжение на выполнение задания

Как найти: Зарплата и кадры → Приказы → Командировка
