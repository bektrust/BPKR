Кадровое перемещение

Документ предназначен для оформления перемещения. Создается на каждого сотрудника отдельно. При выборе Сотрудника на закладке Приказ автоматически заполняются Прежние данные сотрудника, Текущие заполняем вручную.

На закладке Прочие начисления заполняются действующие виды начислений с видом действия Не изменять. Можно вручную установить признак Прекратить.

На закладке Приказ необходимо указать основной вид начисления, на закладке Прочие начисления можно добавить новые строки плановых видов начислений.

Оформить следующий документ Кадровое перемещение возможно после даты возврата сотрудника на предшествующий приказ.

В поле Дата перемещения заполняется дата перемещения сотрудника.

На закладке Дополнительно указывается комментарий к документу.

Как найти: Зарплата и кадры → Приказы → Кадровое перемещение
