Корректировка долга

Документ Корректировка долга предназначен для корректировки взаиморасчетов с контрагентами.
В данном документе предусмотрены следующие виды операций (поле в шапке документа):

  • Перенос задолженности
  • Списание задолженности

Вид операции Перенос задолженности используется для переноса задолженности (дебиторской и/или кредиторской) c одного договора на другой (с одного контрагента на другого, между договорами одного или двух контрагентов).

Вид операции Списание задолженности используется для списания безнадежного долга.

Документом может быть списана одновременно как дебиторская, так и кредиторская задолженность по одному, всем или выбранным договорам контрагента. Для разделения вида задолженности в документе используются соответствующие закладки.

Каждая строка табличной части соответствует одной проводке по переносу или списанию задолженности.

Табличные части заполняются по кнопке «Заполнить остатками по взаиморасчетам» по всем ненулевым сальдо для заданного контрагента (не нужные строки затем удаляются вручную) или вручную по кнопке «Добавить» с выбором договора контрагента.
Внимание:

  • Проводки по операционным курсовым разницам не формируются
  • Расчет НДС на авансы производится в Закрытии месяца (операции могут определяться, как возврат оплаты или возврат отгрузки)
