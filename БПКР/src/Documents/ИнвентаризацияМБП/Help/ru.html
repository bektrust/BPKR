Инвентаризация МБП

Документ Инвентаризация МБП предназначен для проведения инвентаризации МБП, находящихся в эксплуатации.
Если заполнено поле МОЛ - документ заполняется только МБП закрепленных за указанным МОЛ.
Если не заполнено поле МОЛ - заполняются остатками МБП по всей организации (находящимися в эксплуатации).

На закладке МБП заполняются остатки по счетам МБП.

Кнопка Заполнить заполняет табличную часть документа. По данной кнопке реализовано три варианта заполнения:
1. Заполнить по остаткам (МБП) – заполнение документа остатками по МОЛ;
2. Перезаполнить учетные количества и суммы (МБП) – перезаполнение колонок Учетное количество и Учетная сумма;
3. Обнулить фактическое количество (МБП) – обнуление колонки Количество.

В колонке Учетное количество отражается количество по учетным данным в бухгалтерии, эти данные корректировать нельзя.

В колонке Количество заносятся реальные (фактические) остатки по МОЛ. При корректировке данных поля Количество автоматически просчитывается отклонение между реальным остатком, зафиксированным в результате проведения инвентаризации, и остатком по данным учета.

В колонку Сумма заполняется фактическая себестоимость товара. На основании данных, которые указаны в поле Сумма, рассчитываются данные в колонкеЦена.

На закладке Комиссия добавляются члены комиссии.

Документ не формирует проводок.

На основании документа можно оприходовать излишки, которые были выявлены в результате инвентаризации с помощью документа Оприходование товаров или списать недостачу с помощью документа Списание товаров. Их можно создать, нажав на кнопку Создать на основании.

На закладке Дополнительно указывается комментарий к документу.
Так же можно указать данные, для формирования печатных форм:

  • Дата начала и окончания
  • На основании
  • Номер основания
  • Дата основания
  • Причина
  • Комментарий

В документе есть возможность сформировать печатные формы: ИНВ-3 (Инвентаризационная опись товаров), Инвентаризационная опись ИНВ-19.

Как найти: Запасы → МБП → Инвентаризация МБП
