Отпуск

Документ предназначен для оформления отпуска сотруднику. Создается на каждого сотрудника отдельно.

Закладка Средний заработок заполняется автоматически при нажатии кнопки Заполнить, можно добавлять, редактировать строку.

На закладке Начисления по кнопке Рассчитать автоматически рассчитываются начисленные суммы отпускных за указанный период, с разбивкой по видам и месяцам отпуска.

Необходимо заполнить период отпуска сотрудника.

На закладке Дополнительно указывается комментарий к документу.

Печатные формы

Печатные формы

Для документа Отпуск предусмотрены следующие печатные формы:

  • Приказ на отпуск
  • Записка расчет о предоставлении отпуска
  • Справка по расчету отпускных

Как найти: Зарплата и кадры → Табель → Отпуск
