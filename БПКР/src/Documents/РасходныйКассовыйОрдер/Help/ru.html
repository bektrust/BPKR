Расходный кассовый ордер

Документ Расходный кассовый ордер предназначен для учета расхода наличных денежных средств из кассы.

При вводе документа в шапке нужно указать следующие реквизиты, общие для всех операций:

  • Вид операции;
  • Организация – если в программе ведется учет по нескольким организациям;
  • Касса - в справочнике обязательно должен быть указан счет учета и валюта для данной кассы, которые однозначно принимаются для документа Расходный кассовый ордер в целом.

На закладке Расшифровка платежа необходимо указать:

  • Статью ДДС;
  • Сумму платежа.

На закладке Дополнительно необходимо заполнить:

  • Счет учета - устанавливается по выбранной кассе;
  • Счет расчетов - устанавливается из регистра Счета расчетов с контрагентами;

В группе Печать заполняются реквизиты, необходимые для правильного формирования печатной формы документа:

  • Выдать - информация о том, кому выдаются денежные средства из кассы;
  • По документу - документ, удостоверяющий личность физического лица;
  • Основание (для печати) - содержание хозяйственной операции;
  • Приложение - перечисляются прилагаемые первичные и другие документы с указанием их номеров и дат составления;

Если нужно оформить оплату по разным статьям движения денежных средств, то в шапке документа используется кнопка Список.

Курс взаиморасчетов (виден только, если валюта договора и валюта кассы не совпадают) заполняется автоматически, но может быть изменен вручную при договорном курсе.

Сумма взаиморасчетов (видимость как у курса взаиморасчетов) – рассчитывается автоматически. При изменении суммы вручную меняется курс взаиморасчетов.

Оплата поставщику

Операция предназначена для отражения выдачи из кассы организации наличных денежных средств в счет погашения задолженности перед поставщиком по полученным товарам (работам, услугам) или в качестве аванса поставщику в счет предстоящих поставок.

  • Контрагент - поставщик, который является получателем денежных средств;
  • Договор – договор с контрагентом, по которому ведется расчет.

Возврат покупателю

Операция предназначена для выдачи из кассы организации в качестве возврата денежных средств покупателю, ранее полученных от него в случае расторжения договора поставки или пересмотра цены договора в сторону уменьшения.

  • Контрагент - покупатель, который является получателем денежных средств;
  • Договор - договор контрагента, должен иметь вид договора С покупателем.

Расчеты по займам

Операция используется для отражения выдачи наличных денежных средств в счет возврата кредита или займа, полученного от сторонней организации, или предоставления займа, выданного сторонней организации.

Выдача подотчетнику

Операция предназначена для выдачи подотчетному лицу денежных средств.

Взнос в банк

Операция оформляется в том случае, когда происходит получение наличных денег из кассы для последующего их внесения на банковский счет организации. По банку отдельно отражать операцию не требуется.

Выплата ЗП

Операция предназначена для выдачи всей суммы заработной платы к выдаче по ведомости ответственному сотруднику. При этом предполагается, что ответственный сотрудник производит выдачу каждому сотруднику суммы согласно ведомости,
а ведомость с подписями сотрудника в получении сдает в кассу.

Прочий расход

Операция предназначена для оформления расхода денежных средств в случаях, не предусмотренных стандартными операциями, то есть является универсальной. Выбираем операцию Прочий расход денежных средств (с видом операции Прочий расход).

Инкассация

Операция оформляется, если наличные денежные средства из кассы организации вносятся на расчетный счет через инкассаторские службы.

Печатные формы

Для документа Расходный кассовый ордер предусмотрена печатная форма - Расходный кассовый ордер (КО-2).

Ввод на основании

На основании документа Расходный кассовый ордер можно ввести следующие объекты:

  • Поступление товаров и услуг;
  • Поступление в переработку;
  • Приходный кассовый ордер.

Как найти: Банк и касса → Касса → Расходный кассовый ордер
