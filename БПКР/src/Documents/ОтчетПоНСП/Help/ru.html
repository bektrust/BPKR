Отчет по НСП

При формировании отчета указать период в поле Месяц.

Выбрать вид субъекта:

• Малый – формируется за квартал без приложений к отчету;
• Средний – формируется основная форма и приложения к отчету;
• Крупный – формируется отчет за месяц и основная форма.

Выбрать вид отчета:

• Первоначальный
• Уточненный
• Ликвидационный

Сформировать отчет по кнопке Заполнить. В отчете формируются данные по документам:

• Реализация товаров и услуг
• Возврат товаров от покупателя

По кнопке Печать формируются печатные формы отчета:

• Отчет по НСП
• Приложение к отчету по НСП

В печатной форме отчета в строке "В безналичной форме" формируются данные по документам с установленным флажком Безналичный расчет.

На закладке Дополнительно указывается комментарий к отчету.

Если в параметрах учета, в разделе Прочее установить флажок Отчеты в разные ГНС, станет доступным формирование отчетов для разных ГНС. Для этого в справочнике Подразделения необходимо указать для каждого подразделения ГНС, к которому он относится, а также список складов. Затем в каждом документе Реализация товаров и услуг и Возврат товаров от покупателя обязательно заполнение поля Склад. В Отчете по НСП появится поле ГНС, выбрав там нужное ГНС можно сформировать отчет только по реализациям, относящимся к заданному параметру.
