Работа сверхурочно

Документ Работа сверхурочно предназначен для регистрации сверхурочных часов.

В документе предусмотрены варианты оформления на одного или нескольких сотрудников.

На закладке Сотрудники выбрать сотрудника. При выборе сотрудника автоматически заполняются График и Подразделение соответственно приему на работу (или кадровому перемещению) сотрудника, а также появляется поле Валюта, которая заполняется валютой указанной в договоре сотрудника.


Необходимо заполнить поле Вид расчета из справочника Виды начислений, в котором указывается способ расчета Работа сверхурочно и заполняется Коэффициент для расчета сверхурочных

В поле Часов заполнить количество часов, которые сотрудник работал сверхурочно. Указываются для расчета сверхурочных.


В поле Размер автоматически заполняется размер основного вида расчета.

По кнопке Рассчитать автоматически рассчитывается и заполняется поле Результат.

Поле Способ отражения заполняется из справочника Способы отражения заработной платы.

Так же можно указать Причины, обоснование сверхурочной работы.

Кнопка Список отображает табличную часть для заполнения нескольких сотрудников, в которых заполняются все выше указанные реквизиты.

На закладке Дополнительно указывается комментарий к документу Работа сверхурочно.
