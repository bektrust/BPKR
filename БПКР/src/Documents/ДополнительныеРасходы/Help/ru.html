Дополнительные расходы

Документ Дополнительные расходы предназначен для учета услуг сторонних организаций и собственной компании, которые оказывают влияние на себестоимость приобретенных материальных ценностей. Например, в качестве такой услуги может быть, транспортировка или хранение товаров.

В состав документа входят:

• Контрагент- указывается поставщик услуг;
• Договор контрагента- указывается договор с поставщиком, по которому оказывается поступление услуг;
• Сумма доп.расходов (всего);
• От – дата поступления услуг;
• Организация – выбирается автоматически если учет ведется по одной организации; подбирается если учет ведется по нескольким организациям;
• Способ распределения – выбирается способ распределения затрат. Сумма дополнительных расходов может быть распределена на товары и услуги одним из способов:
• По количеству - при выбранном способе распределения затрат, сумма дополнительных расходов распределяется следующим образом: Определяется коэффициент распределения как отношение количества номенклатурной позиции к общему количеству номенклатурных позиций. Полученный коэффициент распределения умножается на сумму расходов и получается сумма дополнительных затрат по каждой номенклатурной позиции;
• По сумме - сумма дополнительных расходов будет распределяться в зависимости от суммы номенклатурных позиций, указанных в табличной части документа. Коэффициент распределения определяется как отношение суммы номенклатурной позиции к общей сумме номенклатурных позиций. Полученный коэффициент распределения умножается на сумму расходов и получается сумма дополнительных затрат по каждой номенклатурной позиции;
• По доп.единицам
Признак Безналичного расчета устанавливается, если оформляется оплата по банку.
• Курс – курс валюты;
• Кратность - указывает, какому количеству денежных единиц базовой валюты соответствует установленный курс.
• Сумма НДС - сумма НДС по услуге;
• Сумма НСП – сумма НСП по услуге;
