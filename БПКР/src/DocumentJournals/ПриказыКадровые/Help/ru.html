Приказы кадровые

Приказы кадровые это журнал документов, где собран перечень всех документов связанных с приемом на работу, кадровым перемещением или увольнением сотрудников.

В Форме списка журнала можно создать документы: Прием на работу, Кадровое перемещение и Увольнение по кнопке Создать.

По кнопке Листа можно создать новый элемент копированием указанного документа, где нужно только указать нового сотрудника.

Для удобства пользователя есть поле, где можно указать сотрудника из справочника Физические лица и отсортировать документы только по выбранному сотруднику.
