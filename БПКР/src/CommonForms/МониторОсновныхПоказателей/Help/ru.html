Монитор основных показателей

Монитор основных показателей предоставляет руководителю информацию в одном отчете по всем основным показателям о состоянии дел в компании.
Из формы монитора можно открыть Отчеты для руководителя, в которых детализированы соответствующие показатели:

  • Остатки денежных средств,
  • Поступление денежных средств,
  • Расход денежных средств,
  • Продажи по контрагентам,
  • Задолженность покупателей,
  • Задолженность поставщикам.
