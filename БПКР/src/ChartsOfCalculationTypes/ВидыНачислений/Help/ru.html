Виды начислений

Список всех видов начислений, предусмотренных в организации, указывается в справочнике Виды начислений.

При создании нового вида начисления необходимо указать Способ расчета, который выбирается из определенного списка и может принимать следующие значения:

  • по месячной ставке по дням – назначается таким видам начисления как Оклад по дням;
  • по месячной ставке по часам – назначается таким видам начисления как Оклад по часам;
  • по часовой ставке – для расчета по заданной часовой ставке и количеству отработанных часов;
  • по дневной ставке – для расчета по заданной дневной ставке и количеству отработанных дней;
  • фиксированной суммой – используется при регистрации разовых начислений, а также начислений, не зависящих от отработанного времени;
  • процентом – может устанавливаться для таких видов начисления как Премия месячная, Доплата за классность и т.д. В этом случае заполняется закладка Базовые виды расчетов в соответствии с которыми рассчитывается база для начисления процентов;
  • работа сверхурочно – при данном способе расчета указываются количества часов и коэффициенты по которому будут оплачиваться сверхурочные часы;
  • выходные и праздничные – назначается для расчета работы в нерабочие дни, указывается коэффициент по которому будут оплачиваться данные дни;
  • по среднему заработку для больничного – может устанавливаться для расчета больничных, где начисление будет вестись по среднему заработку, для этого указывают Период расчета среднего заработка (по умолчанию 3 месяца);
  • по среднему заработку для отпуска – может устанавливаться для расчета отпуска, где начисление будет вестись по среднему заработку, для этого указывают Период расчета среднего заработка (по умолчанию 3 месяца);
  • неявка – используется как вытесняющий вид расчета для учета пропущенных без уважительных причин дней;
  • годовая премия – используется для базовых видов расчета, устанавливается доля для расчета базовой суммы отпускных (%);
  • произвольная формула – можно записать любую формулу по которой работодатель пожелает рассчитывать начисления и удержания.


Для начисления заработной платы от обратного нужно установить флажок Расчет от обратного.


Для начислений, которые рассчитываются процентом (отпуск, больничный лист, по среднему заработку для больничного, по среднему заработку для отпуска, годовая премия) на закладке Базовые виды расчетов необходимо установить базовые виды расчетов, на основании результатов которых будет рассчитано текущее начисление.


У начислений, результаты которых засчитываются в отработанное время, таких как Оклад, необходимо установить признак Зачет отработанного времени (т.е. для фиксированной суммы не важно сколько времени отрабатывал сотрудник, значит флажок не устанавливается).


Для начислений, с которых удерживаются подоходный налог (ПН) и/или социальные отчисления (СФ) устанавливаются признаки Облагается ПН, Облагается СФ.


Для начислений, которые являются базовыми для удержания профсоюзного взноса, необходимо установить признак Начисляются профвзносы.


Для видов расчетов, являющихся дополнительным доходом, устанавливается признак Дополнительный доход. В этом случае вид расчета не включается в начисления, но входит в налогооблагаемые базы.
