Виды удержаний

Список удержаний с заработной платы работников предприятия хранится в справочнике Виды удержаний.


Для каждого нового вида удержания необходимо указать способ расчета:

  • Процентом – указывается размер процента, который будет применен к базе начислений;
  • Фиксированной суммой – указывается сумма ежемесячного удержания.


Для тех видов удержаний, которые будут рассчитываться процентом, необходимо определить базовые виды расчетов из плана видов расчета “Виды начислений”, на основании которых будет рассчитываться текущее удержание.


При необходимости рассчитывать удержания с начислений за минусом налогов устанавливаются соответствующие признаки Расчет за минусом ПН, Расчет за минусом СФ, Расчет за минусом Проф взнос.

В поле Счет учета необходимо выбрать счет для отражений удержания.
