Групповое перепроведение документов

Обработка предназначена для группового перепроведения документов. Например, для восстановления правильной последовательности проведения документов.

Период в обработке задается по кнопкам:

  • Неделя
  • Месяц
  • Квартал
  • Год

После выбора периода справа по кнопке стрелка влево можно уменьшить период, по кнопке стрелка вправо – увеличить период.

Перепровести документы можно по всем организациям или по выбранному списку организаций.

Если учет ведется по нескольким организациям и нужно перепровести документы не по всем организациям, то выбрать настройку По отмеченным организациям и по кнопке Добавить выбрать список организаций.

В случае, если нужно перепровести документы по всем организациям или учет ведется только по одной организации устанавливается настройка По всем организациям.

Перепроведение выполняется по кнопке Выполнить.

Как найти: Операции → Сервис → Групповое перепроведение документов
