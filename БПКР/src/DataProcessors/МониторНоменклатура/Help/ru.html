Монитор номенклатуры

Обработка Монитор номенклатуры предназначен для получения наиболее полной информации об остатках и движениях товара и МБП за период по счетам группы Запасы и МБП.

Для заполнения данных в обработке нужно указать период.

В шапке можно установить отбор по:
• Номенклатуре,
• Счету учета номенклатуры,
• Организации (если учет ведется по нескольким организациям),
• Складу.

Если в программе установлен способ оценки запасов по ФИФО, то в мониторе выводится надпись Учет по ФИФО. Если установлен по средней, то Учет по СВЦ.

Заполняется отчет по кнопке Сформировать.

Настройки в шапке:

Есть три уровня, по которым можно настроить счет, склад и партию (партия для учета по ФИФО). По уровням можно настроить иерархию монитора, например, если первый уровень настроить по складу, а второй по партии, то в мониторе появятся колонки склад и партия. Если не задавать ни один уровень, то формирование будет только по номенклатуре и каждая номенклатура будет выходить по одной строке.

По кнопке Отчеты в шапке можно сформировать отчеты из раздела Запасы.

На закладке Номенклатура в верхней таблице формируются следующие данные:

• Номенклатура,
• Счет,
• Склад,
• Начальный остаток,
• Приход,
• Расход,
• Конечный остаток.

В нижней таблице формируются данные по количеству и сумме, а также выводится цена и склад.

По кнопке Перепровести документы автоматически запускается перепроведение документов за период формирования обработки.

На закладке МБП в верхней таблице формируются следующие данные:

• МБП,
• Счет,
• Склад/МОЛ,
• Начальный остаток,
• Приход,
• Расход,
• Конечный остаток.

В нижней таблице формируются данные по количеству и сумме, а также выводится цена и склад/МОЛ.
