Загрузка ЭСФ

Обработка предназначена для загрузки xml файла электронного счета- фактуры с видом Входящие и последующего создания документов Электронные СФ.

По кнопке Выбрать файл нужно указать путь, к файлу xml, ранее выгруженному из ИС ЭСФ. После выбора файла заполняется форма обработки.
С помощью кнопки Открыть соответствие, можно оформить запись в справочнике Соответствия номенклатуры.

По кнопке Сформировать ЭСФ можно автоматически создать документы Электронные СФ с видом Входящие, в соответствии с загруженными из файла xml данными.
