Монитор ЭСФ


Монитор ЭСФ предназначен для анализа электронных счетов-фактур (за указанный период, по организации):

  • Просмотр документов Электронный СФ с отборами по виду ЭСФ, по наличию номера ЭСФ, по статусу (последнее при наличии api)
  • Просмотр документов, для которых созданы документы ЭСФ. Возможно непосредственное создание документов Электронный СФ из монитора
  • Просмотр по контрагенту и договору


В шапке задаются параметры для отбора документов.

Закладка Контрагенты и Договоры. Содержит перечень Контрагентов, по активному в ТЧ Договоры показывается список договоров с контрагентом, по ним же заполняются поля в шапке Контрагент, Договор.

Закладка Документы. Содержит перечень документов Реализация товаров и услуг, Возврат товаров от покупателя, Электронный СФ в соответствии с заданными параметрами отбора в шапке.

Для документов Реализация товаров и услуг и Возврат товаров от покупателя, на которые не были ранее созданы документы Электронный счет-фактура, с помощью выделения флажком далее по кнопке Создать ЭСФ, можно создать Электронный СФ и здесь же по кнопке Сформировать xml сформировать файл ЭСФ для последующей загрузки в ИС ЭСФ.

Закладка ЭСФ. Формируется информация по текущему документу Электронный СФ на закладке Документы.

Как найти: ЭСФ → Сервис → Монитор ЭСФ
