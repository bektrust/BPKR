Монитор заработной платы

Обработка Монитор ЗП предназначена для просмотра и анализа начислений и удержаний работников по заработной плате.

По кнопке Сформировать заполняются все закладки.

Учетная политика по персоналу – переход в Спр. УПП

Месяц - присваивается автоматически текущий (возможность выбрать вручную месяц).

Подразделение - значение реквизита выбирается из справочника Подразделения

Закладка Сотрудники

Показываются все сотрудники выбранного подразделения, работающие на начало периода и принятые на работу в течение заданного периода. Реквизиты работников показываются на конец периода: Таб.№, Сотрудник, Должность, Подразделение, Документ, Вид события, График работы.

Настройка Сортировка по табельному номеру - при установлении флажка все сотрудники отсортируются по возрастанию по табельному номеру.

По кнопке Добавить можно вручную добавить сотрудников.

Закладка Документы

Показываются все документы начисления ЗП по текущему сотруднику (текущая строка на закладке Сотрудники): начисление зарплаты, закрытие месяца по ЗП, ведомость заработной платы.

Закладка Приказы

Показываются все кадровые приказы по текущему сотруднику (текущая строка на закладке Сотрудники): прием на работу, перемещение, увольнение (независимо от периода).

По кнопке Перепровести документы можно перепровести все приказы кадровые.

Закладка ПНУ

Показываются плановые начисления и удержания по текущему сотруднику за указанный период (только размеры без окончательных расчетов).

Закладка Выплаты

Показываются выплаты по документам ППИ, РКО, АО, а также возвраты (ППВ и ПКО с минусом) по текущему сотруднику за период (3520-11XX, 3520-12XX, 3520-1520, 11хх – 3520, 12хх - 3520).

Закладка Подоходный налог

Подоходный налог с учетом накопления и корректировок.
Каждая строка – месяц. Показываются месяцы с начала года по конец периода.

Настройка Все сотрудники - если флажок установлен, по всем сотрудникам сформируется информация по ПН, если сброшен – по текущему сотруднику, выделенному на закладке Сотрудники.

Закладка Соц отчисления

Соц отчисления с учетом корректировок.
Каждая строка – месяц. Показываются месяцы с начала года по конец периода.

Закладка Расчетный листок

Список начислений и удержаний с указанием данных расчетного листка и с детализацией видов расчета по должностям, подразделениям и приказам.
