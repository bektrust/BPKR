Монитор валютных операций

Видео инструкция: Как пользоваться монитором валютных операций

Монитор предназначен для просмотра итогов и анализа движения по валютным объектам для указанной организации за заданный период.
Валютными объектами могут быть:

  • Счет денежных средств - касса/банк (валюта задается кассой или банк/счетом)
  • Счет расчетов – контрагент - договор (валюта задается договором)
  • Счет расчетов с подотчетниками – подотчетник – валюта

Порядок работы с монитором валютных операций

1. Указать период (месяц) и сформировать
2. На закладке ОСВ (оборотно-сальдовая ведомость) указать анализируемую строку (все суммы в валюте и не зависят от переоценки и закрытия месяца (далее ЗМ)).
Фактически Оборотно-сальдовая ведомость по валютным счетам в разрезе субконто (далее – по субъектам):

  • касса/банк для денежных счетов;
  • контрагент-договор для счетов взаиморасчетов;
  • подотчетник для счета расчетов с подотчетником;

Сортировка Счет – валюта – субконто 1 – субконто 2
Показываются только ненулевые строки (есть хоть одна непустая колонка)
Формирование ОСВ:

  • при изменении периода;
  • при открытии (по сохраненным параметрам);
  • по кнопке сформировать;

3. Закладка «НКР – переоценка»
Таблица по месяцам для текущего субъекта первой закладки за заданный период. Формирование при активизации строки первой закладки.
Неоперационные курсовые разницы формируются документом «Закрытие месяца».
Каждая строка представляет собой месяц - данные до и после закрытия месяца.

На закладке НКР-переоценка:
3.1. Верхняя табличная часть – итоговая строка по НКР

  • Колонка «Сальдо/Расчет» – расчетное сомовое сальдо по проводкам до закрытия месяца
  • Колонка «Сальдо НБ (Нацбанк)» – сальдо по курсу НацБанка
  • Разница между сальдо – это и есть НКР, показывается в колонке «Переоценка»
  • После ЗМ сальдо становится равным Сальдо НБ

3.2. Нижняя табличная часть – расшифровка по документам за период:

  • Сальдо на начало = начальное сальдо в первой строке
  • Сальдо на конец = конечное сальдо в последней строке (до закрытия месяца), то есть для сомов – это расчетное сальдо.

4. Закладка «ОКР»
Таблица проводок по ОКР (операционным курсовым разницам) для текущего субъекта первой закладки за заданный период (проводки со счетами 6280 и 8520). Формирование при активизации строки первой закладки.

Примечание. Операционные курсовые разницы формируются денежными документами (приходный кассовый ордер, расходный кассовый ордер, платежное поручение исходящее, платежное поручение входящее, авансовый отчет и конвертация).
