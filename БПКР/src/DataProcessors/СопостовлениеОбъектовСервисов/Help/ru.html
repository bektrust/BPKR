Сопостовление объектов сервисов

Обработка служит для сопоставления объектов сервисов, загружаемых в 1С.

В открывшейся форме из списка в поле Вид справочника выбрать нужный справочник.
По кнопке Загрузить из файла выбрать вариант загрузки Из внешнего файла - нажать Загрузить таблицу из файла - выбрать файл с перечнем объектов для заполнения справочника, ранее подготовленный для загрузки.

В случае, если наименования объектов справочника, загруженных в 1С совпадают с наименованиями в файле загрузки, то сопоставление произойдет автоматически.
Иначе, необходимо вручную указать соответствующие наименования.

Как найти: ЭТТН → Сервис → Сопоставление объектов сервисов.
