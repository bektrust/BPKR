Выгрузка справочных данных

Для выгрузки данных перейти в Раздел Запасы – Справочник Номенклатура - по кнопке или Еще – Выгрузить данные для ЭСФ. Также можно перейти в Раздел ЭСФ – Сервис – Выгрузка справочных данных.
Перед выгрузкой:

  • для Номенклатуры:

Перед выгрузкой убедиться, что в карточках Номенклатуры заполнены Полное наименование, Единица измерения, Код ТН ВЭД (для товаров), Код ГКЭД (для услуг). Для выгрузки услуг в базе должна быть Единица измерения с наименование «Штука».

  • для Банковских счетов Контрагентов:

Перед выгрузкой убедиться, что в карточках Банковских счетов контрагентов заполнены Наименование, Номер счета, БИК банка, Код валюты.

  • для Покупателей:

Перед выгрузкой убедиться, что в карточках Контрагентов указан ИНН, вид договора «с покупателем».

В поле «Выгружаемые данные» выбрать из списка что необходимо выгрузить и в поле «Имя каталога» указать путь для выгрузки каталога.

Перейти по кнопке Далее и отметить флажками элементы, которые необходимо выгрузить. Выполнить выгрузку по кнопке Выгрузить.

Видео инструкция: ЭСФ. Выгрузка справочника Номенклатура
