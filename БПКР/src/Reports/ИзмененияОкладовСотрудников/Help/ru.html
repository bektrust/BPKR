Изменения окладов сотрудников

Отчет предназначен для формирования ведомости по изменению размеров видов начислений за определенный период времени.

Каждая строка отчета соответствует определенному виду начисления. Все виды начислений сгруппированы по сотрудникам и упорядочены по коду в порядке возрастания.

Для каждого вида начисления показаны новый и прежний размеры.

Как найти: Зарплата и кадры → Очеты → Отчеты по расчету заработной платы → Изменения окладов сотрудников
