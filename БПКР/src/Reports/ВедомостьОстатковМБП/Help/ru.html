Ведомость остатков МБП

Отчет предназначен для формирования ведомости по остаткам малоценных быстроизнашивающихся предметов по всем счетам (субсчетам) за определенный период времени.

Отчет формируется только для МБП введенных в эксплуатацию.

Каждая строка отчета соответствует определенному МБП. Все МБП сгруппированы по подразделениям и МОЛ и упорядочены по наименованию в порядке возрастания.

Для каждого МБП показаны количество и сумма остатка в регламентированной валюте.

Как найти: Запасы → Отчеты → Отчеты по запасам → Ведомость остатков МБП
