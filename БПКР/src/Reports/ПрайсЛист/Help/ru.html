Отчет "Прайс-лист"


Отчет предназначен для просмотра цен на позиции номенклатуры. По умолчанию берутся цены продажи из регистра сведений "Цены номенклатуры документов". При использовании типов цен становится доступным поле "Тип цен" для отображения цен из регистра сведений "Цены номенклатуры".

Флаг "В наличии" выводит только позиции на остатках по следующим счетам:

1) 1600 - Товарно-материальные запасы

2) 1700 - Запасы вспомогательных материалов
