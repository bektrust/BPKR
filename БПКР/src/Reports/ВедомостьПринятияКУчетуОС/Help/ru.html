Ведомость принятия к учету ОС

Отчет предназначен для формирования ведомости по принятию к учету основных средств за определенный период времени.

Каждая строка отчета соответствует определенному основному средству. Все основные средства сгруппированы по счетам учета и упорядочены по наименованию в порядке возрастания.

Для каждого основного средства показаны суммы первоначальной и ликвидационной стоимости.

Как найти: ОС и НМА → Отчеты → Отчеты по основным средствам → Ведомость принятия к учету ОС
