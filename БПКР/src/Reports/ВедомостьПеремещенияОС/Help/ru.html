Ведомость перемещения ОС

Отчет предназначен для формирования ведомости по перемещению основных средств внутри организации по всем счетам (субсчетам) за определенный период времени.

Каждая строка отчета соответствует определенному основному средству. Все основные средства упорядочены по наименованию в порядке возрастания.

Для каждого основного средства показана сумма перемещения в регламентированной валюте.

Как найти: ОС и НМА → Отчеты → Отчеты по основным средствам → Ведомость перемещения ОС
