Расчетные листки

Отчет предоставляет возможность получить сведения о произведенных расчетах по заработной плате организации за указанный месяц.

В отчете формируются следующие данные:
• начисления;
• удержания
• налоги;
• взносы и отчисления;
• сумма выплаченной заработной платы;
• задолженности по заработной плате на начало и конец расчетного месяца.

При формировании отчета выбрать период в поле Месяц.

Поле Подразделение:
1.Если заполнено, то отчет формируется по отдельному подразделению.
2. Если подразделение не указано, то формируется в целом по организации.

Поле Сотрудник:
1. Если заполнено, то отчет формируется по одному сотруднику.
2. Если поле пустое, то формируется по всем сотрудникам организации

Получить данные в отчете по кнопке Сформировать.

Как найти: Зарпалат и кадры → Отчеты → Отчеты по заработной плате → Расчетные листки
