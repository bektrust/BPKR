Штатное расписание

Регистр предназначен для хранения актуальной информации о штатном расписании организации.
Каждому работнику при приеме на работу присваивается должность. Каждой должности соответствует ставка. Отчет Штатное расписание показывает количество свободных и занятых ставок, а также размер оклада соответствующий каждой ставке.

Если в Параметрах Учета установлен флажок Вести штатное расписание:

1. Становиться видимым флажок Контроль соответствия штатному расписанию;
2. Становиться видимым, но не активным флажок Контроль соответствия размера (оклада) штатному расписанию.
3. Ведется штатное расписание: при приеме на работу осуществляется проверка наличия вакантных ставок.

Если в Параметрах Учета установлен флажок Контроль соответствия штатному расписанию:

1. В документах Прием на работу и Кадровое перемещение устанавливается контроль на запрет проведения, если превышаются ставки указанные в штатном расписании.
2. Становиться активным флажок Контроль соответствия размера (оклада) штатному расписанию.

Если в Параметрах Учета установлен флажок Контроль соответствия размера (оклада) штатному расписанию, то в документах Прием на работу и Кадровое перемещение устанавливается контроль на запрет проведения, если размер оклада превышает или меньше размера, указанного в штатном расписании для выбранного вида начисления.

Регистр состоит из двух табличных частей: Подразделение и Должность.

В табличной части Подразделения выводится список подразделений из справочника Подразделение.

В табличной части Должность выходит список должностей при выборе определенного подразделения. Здесь же можно добавлять новые должности при помощи кнопки Создать.


Для каждой записи штатного расписания необходимо указать следующие реквизиты:

• Организация – указывается организация, для которой вводится запись штатного расписания. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
• Подразделение – указывается подразделение, для которого вводится запись штатного расписания. Выбирается из справочника "Подразделение".
• Должность – указывается должность, для которой вводится запись штатного расписания. Выбирается из справочника "Должности".
• Период – указывается период, с которого действует штатное расписание.
• Количество ставок – указывается количество свободных ставок, которое планируется в выбранных организации и подразделении для указанной должности.
• Тариф - Вид начисления, по которому планируется оплачивать работу сотрудников. Выбирается из справочника Виды начислений.
• От – указывается минимальный размер оклада для выбранной ставки.
• До – указывается максимальный размер оклада для выбранной ставки.
