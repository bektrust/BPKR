Оборотно-сальдовая ведомость по парным счетам

Отчет предназначен для формирования оборотно-сальдовой ведомости по выбранному счету и его парному счету за определенный период времени. Парный счет подставляется автоматически.

По составу выводимой информации отчет напоминает фрагмент отчета Оборотно-сальдовая ведомость по счету.

Отчет можно сформировать с детализацией по субсчетам или по объектам аналитического учета (субконто).

Данные можно вывести с дополнительной разбивкой по периодам времени: месяц, год и т.д.

В отчете можно вывести развернутое сальдо. При этом развернутое сальдо рассчитывается для каждого уровня группировки и по паре счетов в целом.

Как найти: Отчеты → Стандартные отчеты → Оборотно-сальдовая ведомость по парным счетам
