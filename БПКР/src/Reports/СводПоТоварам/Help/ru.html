Свод по товарам

Отчет предназначен для формирования сводной ведомости по товарам за определенный период времени.

Каждая строка отчета соответствует определенной номенклатурной позиции. Все номенклатурные позиции сгруппированы по счетам учета и складам и упорядочены по наименованию в порядке возрастания.

Для каждой номенклатурной позиции показаны суммы остатков на начало и окончание периодов, а так же обороты за период. Доплнительно выводится информация о единице измерения.

Как найти: Запасы → Отчеты → Отчеты по запасам → Свод по товарам
