Кадровые изменения

Отчет предназначен для формирования ведомости по кадровым изменениям сотрудников за определенный период времени.

Каждая строка отчета соответствует определенному документу кадрового изменения (прием, перемещение, увольнение). Все документы сгруппированы по сотрудникам и видам кадровых событий и упорядочены по дате в порядке возрастания.

Для каждого вида кадрового события показано количетсво документов.

Как найти: Зарплата и кадры → Очеты → Отчеты по расчету заработной платы → Кадровые изменения
