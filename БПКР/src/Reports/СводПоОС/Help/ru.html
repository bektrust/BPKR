Свод по ОС

Отчет предназначен для формирования сводной ведомости по основным средствам за определенный период времени.

Каждая строка отчета соответствует определенному основному средству. Все основные средства сгруппированы по счетам учета и МОЛ и упорядочены по наименованию в порядке возрастания.

Для каждого основного средства показаны суммы остатков на начало и окончание периодов, а так же обороты за период. Доплнительно выводится информация о дате выпуска, заводском номере и номере паспорта ОС.

Как найти: ОС и НМА → Отчеты → Отчеты по основным средствам → Свод по ОС
