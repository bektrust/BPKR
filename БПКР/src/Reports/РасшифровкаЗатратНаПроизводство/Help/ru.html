Расшифровка затрат на производство

Отчет предназначен для формирования ведомости для расшифровки затрат, свзанных с производством и предназначен для анализа формирования себестоимости.Отчет формируется за определенный период времени.

Каждый строка отчета соответствует определенной статье затрат. Все статьи сгурппированы по подразделениям и номенклатурным группам и упорядочены по наименованию в порядке возрастания.

Для каждой статьи затарт показана сумма данной затраты в регламентированной валюте.

Как найти: Производство → Отчеты → Отчеты по производству → Расшифровка затрат на производство
