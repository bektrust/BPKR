Ведомость начисления заработной платы

Отчет предназначен для формирования ведомости по начисленной заработной плате за определенный период времени.

Каждая строка отчета соответствует определенному сотруднику. Все сотрудники сгруппированы по подразделениям и должностям и упорядочены по наименованию в порядке возрастания.

Для каждого сотрудника показана сумма начисления в регламентированной валюте.

Для каждой должности показано количество специалистов и средниц заработок.

Как найти: Зарплата и кадры → Отчеты → Отчеты по расчету заработной платы → Ведомость начисления заработной платы
