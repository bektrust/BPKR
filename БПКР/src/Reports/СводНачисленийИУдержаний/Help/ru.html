Свод начислений и удержаний

Для того чтобы получить отчет о суммах начислений и удержаний за выбранный период времени с детализацией по видам начислений и удержаний, но без детализации по работникам организации можно воспользоваться отчетом Свод начислений и удержаний.

При формировании отчета указать период формирования отчета.

Получить данные по кнопке Сформировать.

По кнопке Показать настройки можно установить:
1. Отбор по подразделению;
2. Отбор по сотруднику.

По кнопке Печать можно распечатать отчет.

Как найти: Зарпалат и кадры → Отчеты → Отчеты по заработной плате → Свод начислений и удержаний
