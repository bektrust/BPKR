Водители

Для того чтобы заполнить справочник из файла, необходимо открыть обработку Загрузка данных из файла (раздел ЭТТН – Сервис), выбрать справочник Водители из списка и нажать кнопку Далее.

Внимание! Перед загрузкой файла Excel в 1С, необходимо изменить бланк для заполнения в обработке Загрузка данных в справочник Водители:

• «ИНН водителя» на «ИНН»;
• «ФИО водителя» на «ФИО».

Далее выбрать вариант загрузки Из внешнего файла и нажать кнопку Загрузить таблицу из файла. Затем нажать кнопку Записать и закрыть.

Также предусмотрено ручное заполнение справочника. Для этого необходимо нажать на кнопку Создать и заполнить следующие поля:
• ФИО:
• ИНН.

Как найти: ЭТТН → Справочники → Водители
