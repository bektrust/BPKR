Статусы ЗП

Вид занятости сотрудника, а также параметры для расчета налогов хранятся в справочнике Статусы ЗП.

Каждому работнику при приеме на работу присваивается статус. Изменение статуса оформляется документом Приказы кадровые.

В статусе указываются:

Наименование – название статуса.

Категория – категория сотрудников для отчета годовое сообщение.

Вид занятости – вид занятости для формирования отчетов по подоходному налогу.

№ строки в отчете ПН – заполняется, если необходимо вывести суммы по статусу в определенную строку отчета по подоходному налогу.

Метод расчета ПН – определяет какой облагаемый доход нужно будет использовать при расчете ПН. Его можно выбрать из списка:

  • МРД
  • Начислено
  • Начислено больше МРД

Метод расчета СФ – определяет какой облагаемый доход нужно будет использовать при расчете отчислений в СФ. Его можно выбрать из списка:

  • СЗП
  • Начислено
  • Начислено больше СЗП

По кнопке Добавить можно добавить строку в табличную часть, где заполняются данные по налогам вычитаемым с сотрудника и организации. В табличной части заполняются:

Период – указывается дата с которого начинается действие установленных данных.

ПФР – ставка пенсионного фонда с работника;

ГНПФР – ставка государственного накопительного пенсионного фонда с работника (с 1 января 2010);

ПН – ставка подоходного налога;

Вычеты (в РП) – количество стандартных вычетов в расчетных показателях;

ПФФ – ставка пенсионного фонда с организации (фирмы);

МСФ – ставка фонда обязательного медицинского страхования с работодателя;

ФОТФ – ставка фонда оздоровления трудящихся с работодателя (с 1 июля 2010);

Не вычитать ПФ – если признак установлен, то при расчете базы подоходного налога сумма ПФР не вычитается из облагаемых начислений сотрудника.

Как найти: Зарплата и кадры → Справочники → Статусы ЗП
