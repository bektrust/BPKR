Транспортные средства

Для того чтобы заполнить справочник из файла, необходимо открыть обработку Загрузка данных из файла (раздел ЭТТН – Сервис), выбрать справочник Транспортные средства из списка и нажать кнопку Далее.

Внимание! Перед загрузкой файла Excel в 1С, необходимо изменить бланк для заполнения в обработке Загрузка данных в справочник Транспортные средства:

• «Транспортное средство» на «Марка транспорта»;
• «Гос. Номер»» на «Номер транспорта».

Далее выбрать вариант загрузки Из внешнего файла и нажать кнопку Загрузить таблицу из файла. Затем нажать кнопку Записать и закрыть.

Также предусмотрено ручное заполнение справочника. Для этого необходимо нажать на кнопку Создать и заполнить следующие поля:

  • Транспортное средство;
  • Гос. номер.

Как найти: ЭТТН → Справочники → Транспортные средства
