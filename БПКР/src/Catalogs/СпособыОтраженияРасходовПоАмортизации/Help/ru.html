Способы отражения расходов по амортизации

Каждому объекту основных средств необходимо установить способы отражения расходов по амортизации на затраты предприятия по бухгалтерскому учету.

Например, основным средствам, которые используются в производстве, можно указать счет затрат для производственного учета выпускаемой продукции, работ или услуг. Основным средствам, которые используются в административных целях, указать счет затрат, на который будут отнесены амортизационные расходы.

Перечень способов задается в табличной части по кнопке Добавить.

Каждая строка табличной части определяет совокупность параметров для отнесения расходов по амортизации по затратам. В том случае, если введено несколько строк, расходы по амортизации в начале будут распределены по строкам согласно коэффициентам распределения.
Коэффициенты распределения указываются в каждой строке в колонке Коэффициент.

В случае, если в табличной части указана только одна строка, коэффициент следует указывать равный единице.

Как найти: ОС и НМА → Справочники и настройки → Способы отражения расходов по амортизации
