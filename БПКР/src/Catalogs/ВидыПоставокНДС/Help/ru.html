Справочник "Виды поставок НДС"

Справочник предназначен для классификации поставок в целях правильного формирования отчета по НДС. Для каждого документа "Реализация ТМЗ и услуг" на закладке "Счет-фактура" указывается вид поставки НДС.

Справочник состоит только из предопределенных элементов и не редактируется.

Одним из основных предопределенных элементов является "Промышленная переработка с/х продукции". В приложение 1 отчета по НДС попадают поставки по стандартным ставкам НДС (10%, 12%, 20%) с видом поставки НДС, отличным от "Промышленная переработка с/х продукции". Для приложения 2, напротив, отбираются только стандартные поставки номенклатуры с видом поставки "Промышленная переработка с/х продукции".

В приложение 3 попадают все поставки с нулевой ставкой НДС. В зависимости от вида поставки НДС суммы распределяются по графам приложения. Если установлен флажок "Экспорт" на закладке "Счет-фактура" документа "Реализация ТМЗ и услуг", то вид поставки НДС не указывается. за исключением вида "Промышленная переработка с/х продукции".

В приложение 4 попадают все поставки с освобожденной ставкой НДС. Суммы распределяются по графам приложения в зависимости от вида поставки НДС. Если вид поставки НДС не указывается, то поставка попадает в графу "Прочие поставки".
