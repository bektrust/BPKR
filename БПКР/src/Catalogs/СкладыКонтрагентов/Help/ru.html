Склады контрагентов

Для справочника Склады контрагентов нужно выполнить сопоставление с собственными складами, заведенными в ИС ЭТТН. Для этого необходимо:

• Выгрузить файл Excel с перечнем ИНН контрагентов (покупателей) из 1С с помощью обработки Выгрузка справочных данных.

В поле Выгружаемые данные из выпадающего списка выбрать Покупатели. Далее в поле Имя каталога прописать путь для сохранения файла (например, Рабочий стол) и нажать кнопку Далее.

Откроется форма с перечнем контрагентов, с которыми есть вид договора с Покупателем. Нажать кнопку Выгрузить.

  • Загрузить в ИС ЭТТН файл с перечнем Покупателей. После загрузки файла, система сформирует новый файл с перечнем складов покупателей в Excel.
  • Открыть файл и переименовать колонку ПП на ID.
  • Открыть обработку Загрузка данных из файла (раздел ЭТТН – сервис – Загрузка данных из файла).

Выбрать в открывшемся окне справочник Склады контрагентов и нажать кнопку Далее. Выбрать вариант загрузки Из внешнего файла и нажать кнопку Загрузить таблицу из файла. После загрузки файла в табличную часть следует нажать кнопку Загрузить данные.

Следующий шаг – сопоставление складов контрагентов в 1С со складами контрагентов в ИС ЭТТН.

  • Открыть обработку Сопоставление объектов сервисов (раздел ЭТТН – сервис – Сопоставление объектов сервиса).
  • Выбрать Вид справочника – Склады контрагентов и нажать кнопку Загрузить из файла.
  • Выбрать вариант загрузки Из внешнего файла и нажать кнопку Загрузить таблицу из файла.
  • Выбрать файл Excel с перечнем складов покупателей выгруженный ранее из ИС ЭТТН.

Затем нажать кнопку Записать и закрыть. Посмотреть результат сопоставления можно в регистре Соответствие объектов сервиса.

Как найти: ЭТТН → Справочники → Склады контрагентов
