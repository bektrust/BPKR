Справочник "Настройки регламентных отчетов"

1. Справочник предназначен для формирования и хранения настроек соответствующего регламентного отчета.

2. На закладке ОТЧЕТ показаны все строки отчета, включая заголовочные (не рассчитываемые) строки.  Для каждой строки указывается:

    • код строки
    • наименование строки
    • наименование строки на кыргызском языке
    • строка-группа, в которую входит данная строка
    • знак, с которым строка входит в группу (плюс или минус)
    • признак заголовка, если строка заголовочная
    • номер прохода – для указания последовательности расчета групповых строк

    По кнопке НОМЕРА ПРОХОДОВ автоматически формируются номера прохода.

    3. На закладке НАСТРОЙКИ показывается расшифровка для текущей строки закладки ОТЧЕТ – формулы по

    счетам, из которых формируется строка – указываются:

      • код строки
      • счет, сумма по которому входит в строку
      • формула суммы для счета (обороты в дебет, обороты с кредита, сальдо дебетовое конечное, сальдо кредитовое конечное)
      • знак, с котором  сумма счета входит в строку (плюс не показывается)

       

      4. Порядок заполнения справочника:

        • указывается обозначение настройки
        • заполняется список строк отчета на закладке ОТЧЕТ
        • для каждой первичной строки отчета на закладке НАСТРОЙКИ указывается список счетов с формулами и знаками
        • для облегчения заполнения можно заполнить закладки справочника шаблоном по кнопке БАЗОВЫЕ НАСТРОЙКИ и затем отредактировать закладки
