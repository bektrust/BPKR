Статьи движения денежных средств

Справочник предназначен для хранения списка статей движения денежных средств по видам и группам.

Справочник используется для ведения аналитического учета на счетах учета денежных средств для автоматического заполнения формы №4 Отчет о движении денежных средств.

Важно! Корректное заполнение справочника Статьи движения денежных средств влияет на правильность заполнения регламентированного отчета Отчет о движении денежных средств.

Совет. Признак ведения учета по статьям движения денежных средств устанавливается в настройке параметров учета. Если организации не нужно формировать форму №4 Отчет о движении денежных средств, то можно не вести учет денежных средств по статьям движения.

Справочник иерархический и имеет по умолчанию три группы:

  • Инвестиционная деятельность
  • Операционная деятельность
  • Финансовая деятельность

При первом запуске программы производится автоматическое заполнение групп справочника набором стандартных значений. Статья Нереализованные прибыли и убытки от курсовых разниц при этом не входит ни в одну из групп.

В настройках: Параметры учета- Денежные средства можно заполнить следующие параметры для подстановки в денежных документах (ПКО, РКО, ППИ, ППВ):

  • Статья ДДС для оплаты от покупателя
  • Статья ДДС для оплаты поставщику
  • Статья ДДС для прочих поступлений
  • Статья ДДС для прочих выбытий

Статья ДДС (курсовая разница) заполняется автоматически при первом запуске как Нереализованные прибыли и убытки от курсовых разниц.

Статьи ДДС не указываются для операций перемещения денежных средств (касса-касса, банк-банк, касса – банк).
