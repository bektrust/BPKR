Графики работы

Справочник Графики работы содержит перечень всех графиков предприятия и предназначен для целей учета рабочего времени.

Графики настраиваются и заполняются в форме элемента справочника. Представление формы зависит от вида графика. После определения основных параметров в форме графика производится заполнение календаря.

Для заполнения графика необходимо заполнить все параметры (год, период и норму часов) и нажать на кнопку Заполнить.

Порядок заполнения Графика работы:

На закладке Настройка заполнения заполнить следующее:

  • Поле Наименование.
  • В поле Результат заполнения графика на: – указать год, на который заполняется график.
  • Норму часов заполнения – сколько часов положено отработать работнику в рабочий день. Обычно 8 часов.
  • Период в днях – какова периодичность графика, обычно 7 дней (1 неделя). Другое значение может быть, например, для сменных графиков. Не путать с пятидневкой/шестидневкой – это задается в другом поле.
  • Если установлен флажок Учитывать праздники, то в графике (таблица Данные производственного календаря) автоматически будут учтены регламентированные праздники (с точными датами).


После того как заполнены Параметры заполнения по кнопке Записать необходимо записать данные. Затем по кнопке Заполнить заполняется таблица Периодичность:

  • Список рабочих и выходных дней (таблица Периодичность)– заполняется количество строк, равное периоду. Для каждого дня указывается количество рабочих часов (для рабочего дня) и вид дня. Как правило, период – это неделя, то есть 7 дней, при этом возможны другие режимы работы: пятидневка, шестидневка и т.п. Если пятидневка, то будет 5 рабочих дней, суббота и воскресенье. Если шестидневка – 6 рабочих дней, воскресенье. Вид дня нужно указать правильно – по календарю, т.е. если 3 января 2017 было воскресенье, то для даты 03.01.2017 нужно указать вид дня воскресенье. Праздники в списке рабочих и выходных дней не указываются, так как данные заполненного периода будут распространены на весь заполняемый год.
  • После этого нажать на кнопку Заполнить (таблица Данные производственного календаря) – график автоматически заполнится из регламентированного производственного календаря, соответственно таблице Периодичность.


В поле Группа графиков работы можно указать группу, если таковые имеются.

На закладке Результат заполнения можно проверить заполнение календаря на выбранный год.

В поле Результат заполнения графика на – выходит текущий год;

В табличной части отображается Календарь, включающий отображение праздников и выходных дней. При наведении курсора на любую дату из календаря в полях Дней и Часов выводится количество дней и часов в наведенном месяце.

При первоначальном заполнении календаря автоматически учитываются праздничные дни. Отражение в производственном календаре переносов праздников выполняется вручную.

Пользователь может самостоятельно изменять значения этого календаря, щелкнув левой кнопкой мыши по дате и нажав на Изменить день выбрать необходимое значение: рабочий, суббота, воскресение, предпраздничный, праздник, выходной или нерабочий. После внесения всех изменений нужно записать все значения (по кнопке Записать).

При нажатии кнопки Заполнить по шаблону в производственном календаре восстанавливаются все нерабочие дни по регламентированным нерабочим дням (праздникам и выходным).

По гиперссылкам Данные производственного календаря и Периодичность календаря можно увидеть таблицы из закладки Настройка заполнения в развернутом виде.

При помощи кнопки Создать группу можно объединить несколько графиков работы в по группам. Он актуален для больших организаций, где есть филиалы с разными графиками работы.
